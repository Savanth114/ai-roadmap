## 📖 About

**AI Roadmap** is a modern, interactive web platform designed to help students, developers, and AI enthusiasts master Artificial Intelligence through a structured learning path.

Instead of jumping between random tutorials and courses, AI Roadmap provides a clear roadmap that shows **what to learn, when to learn, and how to practice**.

The platform combines roadmap visualization, progress tracking, analytics, gamification, and curated learning resources into one place.

                  ## 🧠 THE COMPLETE 15-PHASE AI LEARNING ROADMAP

--------------------------------------------------------------------------------
## 🟢 PHASE 1: Core Foundations
--------------------------------------------------------------------------------

1. Python Programming
   - Variables and basic data types (strings, integers, floats, booleans)
   - Control flow structures (if-else statements, nested conditions)
   - Loops and iterations (for loops, while loops, comprehension lists)
   - Built-in data structures (lists, dictionaries, tuples, sets)
   - Functions and modular programming (arguments, return values, lambda functions)
   - Object-Oriented Programming (classes, objects, inheritance, polymorphism)
   - File input/output operations (reading and writing text/CSV files)
   - Exception handling (try-except blocks, raising custom errors)
   - NumPy library (multi-dimensional arrays, vectorization, indexing)
   - Pandas library (DataFrames, data cleaning, filtering, grouping)
   - Scikit-learn Library (Data preprocessing, pipelines, classical ML model training)
   - Matplotlib & Seaborn Libraries (Data visualization, statistical plotting)
   - PyTorch Library (Tensors, computation graphs, deep learning building blocks)
   - Hugging Face Transformers Library (Loading pre-trained open-source LLMs and tokenizers)

2. Mathematics for AI
   - Linear Algebra (matrices, vectors, matrix multiplication, transposes)
   - Vector spaces (eigenvalues, eigenvectors, dot products, cross products)
   - Matrix decomposition (Singular Value Decomposition, PCA math)
   - Differential Calculus (derivatives, partial derivatives, chain rule)
   - Optimization mathematics (gradient descent, local minima, saddle points)
   - Probability theory (conditional probability, Bayes' theorem, independence)
   - Probability distributions (Gaussian, Bernoulli, Binomial, Poisson)
   - Descriptive statistics (mean, median, variance, standard deviation)
   - Inferential statistics (hypothesis testing, p-values, confidence intervals)

3. Artificial Intelligence (AI) Foundation
   - History and evolution of AI (Turing test, symbolic era, connectionist era)
   - Symbolic AI (rule-based expert systems, knowledge representation)
   - Search algorithms (uninformed search, breadth-first search, depth-first search)
   - Heuristic search (A* algorithm, greedy best-first search)
   - Adversarial search (Minimax algorithm, Alpha-Beta pruning)
   - Constraint Satisfaction Problems (backtracking search, constraint propagation)
   - Propositional and first-order logic (inference rules, resolution)

--------------------------------------------------------------------------------
## 🔵 PHASE 2: Data & Core Modeling
--------------------------------------------------------------------------------

4. Database Systems
   - Relational database concepts (tables, primary keys, foreign keys)
   - Structured Query Language (SQL CRUD operations, joins, aggregations)
   - Database normalization (1NF, 2NF, 3NF, BCNF)
   - Indexing and query performance tuning
   - NoSQL Document stores (MongoDB schema-free architecture, querying)
   - NoSQL Key-Value stores (Redis caching mechanisms)
   - Columnar databases (Apache Cassandra architecture)
   - Database transactions (ACID properties, isolation levels)

5. Machine Learning (ML) & Optimization
   - Supervised learning paradigms (regression versus classification)
   - Linear and Logistic Regression (cost functions, ordinary least squares)
   - Decision Trees and Ensemble methods (Random Forests, Gradient Boosting, XGBoost)
   - Support Vector Machines (linear margins, kernel trick)
   - Unsupervised learning paradigms (clustering versus dimensionality reduction)
   - Clustering algorithms (K-Means, Hierarchical clustering, DBSCAN)
   - Dimensionality reduction techniques (Principal Component Analysis, t-SNE)
   - Hyperparameter tuning methods (Grid search, Random search, Bayesian optimization)
   - Overfitting mitigation (L1/L2 regularization, bias-variance tradeoff)
   - Performance evaluation metrics (accuracy, precision, recall, F1-score, ROC-AUC)

6. Data Engineering & Pipelines
   - ETL and ELT architectural design patterns
   - Data ingestion techniques (batch processing versus real-time ingestion)
   - Workflow orchestration engines (Apache Airflow DAGs, scheduling)
   - Modern data warehousing structures (Snowflake data sharing, Google BigQuery)
   - Data Lakehouses (Delta Lake transaction layers, Apache Iceberg format)
   - Operational feature management (Feast feature store setup)
   - Data lineage tracking and schema evolution management

--------------------------------------------------------------------------------
## 🟣 PHASE 3: Deep Learning & Linguistics
--------------------------------------------------------------------------------

7. Deep Learning (DL) & Neural Networks
   - Artificial Neuron mechanics (perceptrons, linear combinations)
   - Activation functions (Sigmoid, Tanh, ReLU, Leaky ReLU)
   - Forward propagation and network graph computation
   - Backpropagation mechanics (computational graphs, error gradients)
   - Loss optimization algorithms (Stochastic Gradient Descent, RMSprop, Adam)
   - PyTorch framework fundamentals (tensors, autograd, custom neural network modules)
   - Regularization techniques for deep networks (Dropout layers, Batch Normalization)

8. Natural Language Processing (NLP) & Audio
   - Text tokenization and linguistic normalization (stemming, lemmatization)
   - Static word embedding generation (Word2Vec CBOW/Skip-gram, GloVe structures)
   - Sequence-to-Sequence (Seq2Seq) model frameworks
   - Audio signal preprocessing (sampling rates, waveforms, Fast Fourier Transforms)
   - Mel-Frequency Cepstral Coefficients (MFCC computation for audio models)
   - Automatic Speech Recognition (ASR acoustic modelling patterns)
   - Text-to-Speech (TTS acoustic synthesis workflows)

9. Large Language Models (LLMs) & Generative AI
   - Self-Attention mechanism math (Query, Key, Value vector computation)
   - Multi-Head Attention and Transformer layer stacking
   - Foundational encoder-decoder architectures (BERT processing, GPT scaling laws)
   - Prompt design methodologies (Chain-of-Thought, Few-Shot, Tree-of-Thoughts)
   - Retrieval-Augmented Generation (RAG architecture, context indexing)
   - Vector search indexing (Cosine similarity, Dot product matching, HNSW indexing)
   - Parameter-Efficient Fine-Tuning (PEFT, Low-Rank Adaptation/LoRA frameworks)
   - Deep quantization paradigms (BitsAndBytes implementation, 4-bit/8-bit precision)

--------------------------------------------------------------------------------
## 🟠 PHASE 4: Advanced Autonomy & Systems
--------------------------------------------------------------------------------

10. Agentic AI & Autonomy
    - Core autonomous agent architecture (perception, memory, action loop)
    - Reasoning and planning frameworks (ReAct prompt loops, Plan-and-Solve)
    - Multi-agent orchestration systems (CrewAI sequential tasks, LangGraph states)
    - Digital tool utilization frameworks (dynamic function calling, external API execution)
    - Long-term agent memory management (semantic memory buffers, episodic logging)
    - Autonomous goal breakdown (recursive task decomposition loops)

11. Reinforcement Learning (RL)
    - Markov Decision Processes (states, action sets, transition probabilities)
    - Bellman equations and value functions
    - Value-based optimization models (Q-Learning tables, Deep Q-Networks/DQN)
    - Policy Gradient optimization methods (Proximal Policy Optimization/PPO)
    - Exploration versus exploitation balancing (Epsilon-greedy, Upper Confidence Bound)
    - Reinforcement Learning from Human Feedback (RLHF reward model tuning patterns)

12. Fuzzy Logic & Hybrid Systems
    - Fuzzy Set Theory concepts (membership functions, crisp versus fuzzy boundaries)
    - Fuzzy logical operators (fuzzy intersections, unions, complements)
    - Fuzzy Inference Systems (Mamdani system workflows, Sugeno system models)
    - Defuzzification methodologies (centroid method, mean of maximum)
    - Adaptive Neuro-Fuzzy Inference Systems (ANFIS structural parameters)
    - Hybrid neuro-fuzzy system optimization loops

--------------------------------------------------------------------------------
## 🔴 PHASE 5: Infrastructure, Cloud & Governance
--------------------------------------------------------------------------------

13. Docker Containerization
    - Containerization versus Virtual Machine structural architectures
    - Dockerfile development (base images, layered builds, environment exposures)
    - Docker image storage and version tagging parameters
    - Docker container life cycle operations (run, stop, inspect, exec commands)
    - Docker multi-container configurations (Docker Compose orchestration scripts)
    - Docker networking infrastructure (bridge systems, overlay definitions)
    - Container data preservation (named volumes, bind mounts)

14. Cloud Computing with AWS
    - AWS Identity and Access Management (IAM users, roles, resource access policies)
    - Amazon Elastic Compute Cloud (EC2 instance scaling, GPU instance types)
    - Amazon Simple Storage Service (S3 bucket policies, storage tiering structures)
    - AWS Lambda function deployment (serverless triggers, container images)
    - Amazon Relational Database Service (RDS setups, scaling configurations)
    - Amazon DynamoDB architecture (NoSQL primary keys, secondary indexes)
    - AWS SageMaker workspaces (hosted notebook compute instances, training endpoints)
    - AWS Bedrock integrations (managed serverless access to foundation LLMs)

15. AI Governance, Safety & Trust
    - Explainable AI (XAI diagnostic mechanisms, SHAP values, LIME frameworks)
    - Algorithmic bias classification and demographic parity testing methods
   - System safety protection mechanisms (NeMo Guardrails, Llama Guard inputs)
   - Data privacy enhancement applications (Differential privacy, Homomorphic encryption)
   - Regulatory legislative standard reviews (EU AI Act compliance metrics, NIST)
